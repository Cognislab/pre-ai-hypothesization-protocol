# AIに聞く前の5分間
## Pre-AI Hypothesization Protocol v0.1

Pre-AI Hypothesization Protocol は、生成AIに質問する前に、ユーザー自身の仮説・判断基準・確認点を短く書き出すための小型プロトコルです。

本プロトコルは、生成AIの利用を避けるためのものではありません。また、「AIを使うと考える力が落ちる」と断定するものでもありません。

生成AIをよりよく使うために、AIに質問する前の自分の見立てを短く残し、AIとの対話の中で自分の考えがどう変わったのかを見えるようにすることを目的とします。

## 目的

生成AIとの壁打ちは、考えを整理する有効な方法になりつつあります。

一方で、AIの応答に入る前の「自分は何をどう見ていたか」は、対話の中で見えにくくなることがあります。

本プロトコルでは、AIに質問する前に次の3点を短く書き出します。

1. いま自分はどう見ているか
2. 何を基準に判断するか
3. どこを確認すべきか

これにより、AI応答後に「何が変わったのか」「何が残ったのか」を見返しやすくします。

## 基本的な使い方

1. 新しいAIチャットを開く
2. `prompt_ja.md` の内容を貼り付ける
3. モードとテーマを入力する
4. AIの問いに答える
5. AIの補助回答を受け取る
6. 最後に、自分の判断・違和感・変化を一文で書き留める

## 4つのモード

- 判断モード：買う／やる／投稿する／選ぶ
- 理解モード：ニュース・論文・制度・文章を読む
- 創作モード：俳句・記事・タイトル・小説案
- 検証モード：AI回答の主張や事実性を確認する

## 注意

本プロトコルは、思考力の向上を実証するものではありません。

生成AI利用時に、ユーザー自身の仮説・判断基準・確認点を対話内に残すための実験的な補助線です。
